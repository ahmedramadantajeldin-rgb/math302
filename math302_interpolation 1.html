§ 01

Introduction to Interpolation

Definition & Purpose

Interpolation is the process of constructing a continuous function — an interpolating polynomial — that passes exactly through a given set of discrete data points \((x_0, y_0), (x_1, y_1), \ldots, (x_n, y_n)\).

The resulting polynomial \(P_n(x)\) can then be used to estimate function values between the known data points, compute derivatives, and evaluate definite integrals — tasks that are essential throughout engineering analysis.

الاستيفاء (Interpolation) هو إيجاد دالة متصلة تمر بالضبط عبر مجموعة من النقاط المعطاة، تُستخدم هذه الدالة لتقدير قيم مجهولة وحساب المشتقات والتكاملات عدديًا. أبرز أساليبه: طريقة نيوتن للفروق المنقسمة، وطريقة لاغرانج.
Three Core Applications
1
Estimation: Predict unknown values \(f(x^*)\) for \(x^*\) between data nodes — the fundamental use case.
2
Numerical Differentiation: Compute the rate of change \(\frac{d}{dx}P_n(x)\big|_{x=x^*}\) when the closed-form derivative of the original function is unavailable.
3
Numerical Integration: Evaluate \(\int_a^b P_n(x)\,dx\) as an approximation to \(\int_a^b f(x)\,dx\) — the basis of Newton–Cotes formulas.
Newton's Divided Difference — Method Overview

Newton's Divided Difference (NDD) method constructs an interpolating polynomial incrementally using a triangular table of divided differences. The polynomial takes the form:

General Form — Newton's Interpolating Polynomial \[P_n(x) = a_0 + a_1(x - x_0) + a_2(x-x_0)(x-x_1) + \cdots + a_n(x-x_0)(x-x_1)\cdots(x-x_{n-1})\]

where the coefficients \(a_k\) are the k-th order divided differences evaluated at the leading nodes:

Coefficient Definition \[a_0 = f[x_0],\quad a_1 = f[x_0,x_1],\quad a_2 = f[x_0,x_1,x_2],\quad \ldots\] \[f[x_i, x_{i+1}] = \frac{f[x_{i+1}] - f[x_i]}{x_{i+1} - x_i}\]
المعاملات \(a_k\) هي قيم الفروق المنقسمة المرتبة من الجدول المثلثي. كلٌّ منها يُحسب بقسمة الفرق بين فرقَين من المرتبة الأدنى على الفرق بين نقطتي البداية المقابلتين.
Truncation Error Bound

The error of an \(n\)-th degree interpolating polynomial at any point \(x\) satisfies:

Error Bound Formula \[\left|f(x) - P_n(x)\right| \le \left|\frac{(x-x_0)(x-x_1)\cdots(x-x_n)}{(n+1)!}\right| \cdot \left|f^{(n+1)}(\xi)\right|, \quad \xi \in [\min x_i,\; \max x_i]\]

When \(f^{(n+1)}\) is unknown, the expression \(\dfrac{|(x-x_0)\cdots(x-x_n)|}{(n+1)!}\) gives the error multiplier — the bound in terms of the unknown higher derivative.

Problem 1 — Fourth-Degree Interpolation P₄(x)

01
Forward Divided Difference · 5 Data Points Find P₄(x); compute the max error at x = 2.5, the rate of change, and the area under the curve from x = 0 to x = 4.
Given Data
\(x\)\(x_0=0\)\(x_1=1\)\(x_2=2\)\(x_3=3\)\(x_4=4\)
\(y = F(x)\) 01341
(A) Conceptual Explanation

We have \(n+1 = 5\) equally-spaced points, so we build a 4th-order polynomial. The divided difference table is constructed column by column: each entry is the difference of its two neighbors above divided by the horizontal span of their \(x\)-values.

Construction Rule \[f[x_i, \ldots, x_{i+k}] = \frac{f[x_{i+1},\ldots,x_{i+k}] - f[x_i,\ldots,x_{i+k-1}]}{x_{i+k} - x_i}\]

The diagonal entries of the table — \(f[x_0], f[x_0,x_1], f[x_0,x_1,x_2],\ldots\) — become the polynomial coefficients \(a_0, a_1, a_2, \ldots\)

Problem 2 — Cubic Interpolation P₃(x)

02
Newton's Divided Difference · 4 Data Points Find P₃(x); compute max error at x = −2.5, rate of change at x = −2.5, and area from x = 0 to x = 3.
Given Data
\(x\)\(x_0=0\)\(x_1=-1\)\(x_2=-2\)\(x_3=-3\)
\(y = F(x)\) 010714
(A) Conceptual Explanation

Four data points determine a unique polynomial of degree at most 3. Note the non-uniform spacing: the \(x\)-nodes decrease by 1 from 0 to −3, so the denominators in the divided differences are consistently \((-1)\), \((-2)\), and \((-3)\) respectively.

Key Observation — Negative Spacing When nodes are listed in decreasing order, divided differences involve negative denominators. The divided difference formula remains identical; only the sign of the denominator changes. \[f[x_0, x_1] = \frac{f[x_1] - f[x_0]}{x_1 - x_0} = \frac{10 - 0}{-1 - 0} = -10\]
لاحظ أن نقاط البيانات مرتبة تنازليًا من 0 إلى −3، لذا فإن مقامات الفروق المنقسمة ستكون سالبة. هذا لا يغيّر الطريقة، بل يؤثر فقط على إشارة المعاملات. المعامل \(a_1 = -10\) ناتج عن هذا الترتيب.

Lagrange Interpolation — Verification

Method Overview

The Lagrange form expresses the interpolating polynomial as a weighted sum of basis polynomials \(L_i(x)\), each constructed to be 1 at node \(x_i\) and 0 at all other nodes:

Lagrange Interpolating Polynomial \[P_n(x) = \sum_{i=0}^{n} y_i\, L_i(x), \qquad L_i(x) = \prod_{\substack{j=0\\j\ne i}}^{n}\frac{x - x_j}{x_i - x_j}\]

The Lagrange and Newton forms are algebraically identical — they represent the same unique polynomial of degree \(\le n\) through \(n+1\) points. Verifying one via the other confirms the result.

طريقة لاغرانج والطريقة النيوتنية يُنتجان نفس كثير الحدود تمامًا — هذا مضمون بنظرية تفرّد كثير الحدود الاستيفائي. التحقق بطريقة لاغرانج يُعزّز الثقة بالنتيجة ويُثبت عدم وجود خطأ حسابي.
Lagrange Basis Polynomials for Problem 2

Using nodes \(x_0=0,\; x_1=-1,\; x_2=-2,\; x_3=-3\) and values \(y = 0, 10, 7, 14\):

L₀(x) — basis at x₀ = 0 \[L_0(x) = \frac{(x+1)(x+2)(x+3)}{(1)(2)(3)} = \frac{1}{6}(x^3+6x^2+11x+6)\]
L₁(x) — basis at x₁ = −1 \[L_1(x) = \frac{(x)(x+2)(x+3)}{(-1)(1)(2)} = -\frac{1}{2}(x^3+5x^2+6x)\]
L₂(x) — basis at x₂ = −2 \[L_2(x) = \frac{(x)(x+1)(x+3)}{(-2)(-1)(1)} = \frac{1}{2}(x^3+4x^2+3x)\]
L₃(x) — basis at x₃ = −3 \[L_3(x) = \frac{(x)(x+1)(x+2)}{(-3)(-2)(-1)} = -\frac{1}{6}(x^3+3x^2+2x)\]
Assembling the Lagrange Polynomial
Full Lagrange Sum \[P_3(x) = (0)\,L_0(x) + (10)\,L_1(x) + (7)\,L_2(x) + (14)\,L_3(x)\] \[= -5(x^3+5x^2+6x) + \frac{7}{2}(x^3+4x^2+3x) - \frac{7}{3}(x^3+3x^2+2x)\]

Collecting coefficients of each power:

Result — identical to Newton form ✓ \[P_3(x) = -\frac{23}{6}x^3 - 18x^2 - \frac{145}{6}x\]
Verification Conclusion Both Newton's Divided Difference and Lagrange methods produce the same polynomial \(P_3(x)\). This confirms the uniqueness of the interpolating polynomial and validates both computations.
